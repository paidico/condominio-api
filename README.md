condominio-api
==============

API para gestão de condomínio
-----------------------------

Disponível em https://github.com/paidico/condominio-api.git

* Pré-requisitos:
  * [node.js](http://nodejs.org/)
  * [mongoDB](https://www.mongodb.org/)
* Inicialização:
```SH
$ npm install
$ node server
```
